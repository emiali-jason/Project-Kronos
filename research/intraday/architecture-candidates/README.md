# Architecture Candidates

This directory contains unapproved, technology-neutral, parameter-free candidate principles. Use the template in [`../templates/architecture-candidate-template.md`](../templates/architecture-candidate-template.md).

Presence here never authorises an architectural or production change.

See the [candidate index](INDEX.md) for current review objects. The [provisional responsibilities note](PROVISIONAL-engine-responsibilities.md) is a research synthesis, not a final Engine Map.

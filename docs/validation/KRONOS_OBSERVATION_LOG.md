# KRONOS Observation Log

## Purpose

This log records notable chart behaviour, KRONOS panel output, independent chart readings, and follow-up evidence for future engineering decisions. It is a research record, not an immediate change request list.

One observation does not justify one code change. A change should normally require repeated evidence unless the observation reveals a clear runtime or calculation defect.

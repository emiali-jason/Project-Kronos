# KRONOS Options Research Knowledge Base

**Status:** Draft Research  
**Classification:** Research / Working Notes  
**Phase:** Phase 1 — Evidence Repository  
**Product:** KRONOS Options  
**Architecture Status:** Not approved  
**Authority:** None

## Purpose

This directory preserves options research for Project KRONOS. It is an additive evidence area outside `docs/architecture/`; inclusion in the repository does not promote any content into approved architecture.

KRONOS Options remains a future product. No Options engine, product architecture, ownership model, interface, ADR, trading rule, threshold, strike-selection formula, or implementation is approved here.

The latest explicit task authorizes this repository research index. The accepted Google Drive working notes remain source artifacts; their appearance in the evidence matrix does not change their original research-only classification.

## Boundaries

- Evidence must precede architecture.
- Observation, evidence, possible interpretation, and open questions remain separate.
- Source claims are not KRONOS conclusions.
- Exact market levels, performance claims, ratios, probabilities, and source-specific rules remain unvalidated unless an authoritative source establishes a contract mechanic.
- Copyrighted PDFs, ebook files, copied transcripts, long book passages, trading recommendations, and code are prohibited.
- Research does not own or redefine Discovery, Intraday, Swing, Portfolio, Execution, Engineering, or any approved engine.
- Approved architecture under [`../../architecture/`](../../architecture/) is unchanged and remains authoritative.

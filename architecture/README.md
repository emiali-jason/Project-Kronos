# Architecture

This area is reserved for approved or actively reviewed system design, engine boundaries, constitutions, information flow, decision philosophy, and architecture decisions. Architecture must not contain implementation code.

The repository's existing authoritative architecture documents remain in [`docs/architecture/`](../docs/architecture/) and [`docs/ARCHITECTURE.md`](../docs/ARCHITECTURE.md). They have not been moved or duplicated. New material belongs here only through explicit architecture review.

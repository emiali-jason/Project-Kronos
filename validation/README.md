# Validation

This area is for validation observations, experiments, evidence, behavioural reviews, expected-versus-actual behaviour, and architecture validation. Validation must not directly change production behaviour.

Existing validation guidance remains in [`docs/validation/`](../docs/validation/), recorded evidence remains in [`data/validation/`](../data/validation/), and validation images remain in [`assets/validation/`](../assets/validation/). Those established locations have not been moved or duplicated.

# Extracted Principle Consensus

This directory records agreement, conflict, and unanswered questions across registered research sources for each existing Extracted Principle.

Consensus records are research synthesis only. They do not validate or accept an Extracted Principle, approve an Architecture Candidate, create architecture, or authorise implementation. Source counts measure traceability coverage, not truth or evidence independence. Repetition of the same claim or reuse of the same underlying information does not strengthen evidence by itself.

See the [Consensus Index](INDEX.md), [Extracted Principle Index](../extracted-principles/INDEX.md), and [Evidence Classification](../governance/evidence-classification.md).
